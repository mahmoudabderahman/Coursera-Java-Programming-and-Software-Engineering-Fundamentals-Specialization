####################################################################
                                STRINGS
####################################################################
Some important String methods:
 - The substring() method includes the first parameter and excludes the second parameter (s.indexOf("s")).
 - The indexOf() method returns the index of the first occurrence of a character in a string (s.indexOf("g")).
        - If a second parameter is provided, the search will begin by this index.
 - The startsWith() method returns true or false stating if the string has this passed string as a first string.
 - The endsWith() method returns true or false stating if the string has this string as a end string.
